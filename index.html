🧪Pharmcs

Projeto de Pesquisa e Inovacão do 1 SEM - SPTECH School 👨‍🎓

Pharmcs é uma solução para o monitoramento de umidade e temperatura de geladeiras farmacêuticas.

Foi utilizado o Arduino para realizar o monitoramento simulando um sensor acoplado as geladeiras onde são realizadas as medições de temperatura e umidade e apartir disso ter alertas e notificações para o cliente.

Git para versionamento de código, Github para armazenamento do código em um repositório remoto, C++ para escrita dos scripts do Arduino, MySQL como banco de dados, HTML / CSS / JavaScript para elaboração de site institucional com login e cadastro para acesso á um dashboard com métricas e alertas conectado a uma api em Node.js, realizando a gestão do projeto com a metodologia ágil Scrum e Kanban com sprints semanais organizadas no Notion.
